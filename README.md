## Overview
This project is a basic Hangman game made with **Python** and **Pygame**. It was built as a group project to practice coding, teamwork, and game design. The game lets players guess letters to find a hidden word, with a stick figure drawn step by step for wrong guesses.

## How to Play
- The computer picks a random word.
- Players guess letters using the keyboard.
- Correct letters appear in the word.
- Wrong guesses add parts to the hangman drawing.
- The game ends when:
  - The word is guessed (win), or
  - The hangman is fully drawn (lose).

## Features
- Simple graphics using Pygame.
- Easy keyboard input for guessing letters.
- Visual feedback for correct and wrong guesses.
- Fun and quick to play.
